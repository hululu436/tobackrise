# 攀大指南

## 一. 介绍

* 大学生活丰富多彩，我们会在这里得到各种各样的机会机遇，但是我们的精力却是有限的。

* 大多数人都有适合自己的抉择方法，而明智的抉择方法都是基于对事物的准确认识。

这个版块基于以上两点客观认识，提供尽量客观准确的校园生活、学习、竞赛等方面的详尽介绍、评价、推荐。

众所周知，一个人的认识是不可能绝对客观的，所以这个版块欢迎任何校友、老师贡献自己对某些方面的认识，以及经验观点。
